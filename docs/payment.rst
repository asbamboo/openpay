支持的支付渠道
==============================

渠道名称      支持openpay版本     代码库                            
============ ================= =================================
支付宝支付      v1.0              `asbamboo/openpay-alipay`_      
微信支付        v1.0              `asbamboo/openpay-wxpay`_
============ ================= =================================
